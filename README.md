# paintforlife

## Description
Website about selling paints and paint supplies. Users can browse products, add items to cart, search for paints, apply promo codes, and place orders.

## Technologies Used
- **Frontend**: HTML5, CSS3, Bootstrap 5
- **JavaScript**: Vanilla JavaScript (ES6+)
- **Storage**: LocalStorage for cart persistence
- **Backend**: Python Flask (for order submission)

## Features
- Product catalog (paints and spray cans)
- Shopping cart with persistent storage
- Search functionality
- Product filtering by type
- Promo code system (PAINT20 for 20% discount)
- Order form with validation
- Chatbot assistant
- Responsive design (mobile-friendly)
- Notification system

## Installation
1. Clone the repository
2. Open `index.html` in a modern web browser
3. Ensure Flask backend is running on `http://127.0.0.1:5000` for order submission

## Usage
- Browse products in the main page
- Click "Add to Cart" to add items
- Click cart button to view and manage cart
- Apply promo code "PAINT20" for discount
- Fill order form and submit
- Use search bar to find specific paints
